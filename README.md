## ğŸ“‹ Setup Instructions

### 1. Clone Repository
```bash
git clone https://github.com/rsys-spsingh/amity-online-university.git
cd amity-online-university
```

### 2. Deploy to GitHub Pages
1. Go to repository Settings
2. Navigate to "Pages" section
3. Select "Deploy from a branch"
4. Choose "main" branch
5. Save settings

### 3. Configure Custom Domain
1. Ensure `CNAME` file contains: `amityonlineadmission.in`
2. In repository Settings > Pages
3. Add custom domain: `amityonlineadmission.in`
4. Enable "Enforce HTTPS"

### 4. DNS Configuration (Hostinger)
Add these DNS records in your Hostinger control panel:

